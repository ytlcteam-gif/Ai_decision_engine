## Vercel Environment Variables

Add the following variable in the Vercel Dashboard under Project Settings > Environment Variables:

- `GEMINI_API_KEY` - your Gemini API key. Keep this server-side only.

No `NEXT_PUBLIC_` variables are required for the current app. The frontend calls the local API route directly, so no custom `vercel.json` rewrites are needed.
Check out our [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
